# 🚀 Employee CRM System
A multi-page Employee CRM system built using HTML, CSS, JavaScript, Bootstrap, and jQuery.

## 🔹 Features
* Employee Management Dashboard
* Skill Assessment System
* Dynamic Profile Page
* LocalStorage Integration
* Modern Glass UI Design

## 🛠 Technologies Used
* HTML5
* CSS3
* Bootstrap 5
* JavaScript (ES6)
* jQuery

## 📂 Modules
* Dashboard
* Employees
* Assessment
* Profile

## 🎯 Highlights
* Fully client-side application
* Real-time data using LocalStorage
* Clean and responsive UI
